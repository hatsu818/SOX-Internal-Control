## 📋 プロセス概要情報

| 項目 | 内容 |
|------|------|
| **プロセス目的** | 固定資産の取得において適切な承認手続きと正確な会計処理を実施し、資産の不正取得を防止し財務報告の信頼性を確保する |
| **関連業務** | 購買業務、会計処理業務、資産管理業務 |
| **関連部門** | 総務部、経理部、購買部、各事業部門、情報システム部 |
| **関連システム** | 資産管理システム、購買管理システム、会計システム |
| **頻度** | 都度（固定資産取得の都度発生） |
| **重要性** | 高（財務報告への直接的影響） |
