## 🔍 アサーション別詳細分析

### 実在性（Existence）
- **R-001**: 架空売上の計上リスク → C-001で注文内容確認
- **R-002**: 与信管理不備 → C-002で与信限度チェック  
- **R-003, R-004, R-005**: 物理的な商品の存在確認 → C-003, C-004, C-005で照合
- **R-006**: 売上の実在性確認 → C-006で出荷実績照合
- **R-007, R-008, R-009**: 債権の実在性確認 → C-007, C-008, C-009で金額・残高確認

### 網羅性（Completeness）
- **R-002**: 全ての与信情報の把握 → C-002で包括的チェック
- **R-003**: 全ての在庫情報の把握 → C-003で在庫データ照合
- **R-006**: 全ての売上取引の記録 → C-006で売上計上の網羅性確認
- **R-008, R-009**: 全ての入金の記録 → C-008, C-009で入金・消込の網羅性確認

### 権利義務（Rights & Obligations）
- **R-001**: 売上に対する権利の適切性 → C-001で取引実態確認
- **R-002**: 債権回収権利の確保 → C-002で与信管理
- **R-006**: 売上認識権利の適切性 → C-006で出荷基準確認
- **R-008, R-009**: 債権の法的権利確認 → C-008, C-009で債権管理

### 評価測定（Valuation & Allocation）
- **R-002**: 貸倒引当金の適切な評価 → C-002で与信評価
- **R-007**: 売上金額の正確な測定 → C-007で請求金額確認

### 期間帰属（Cut-off）
- **R-001**: 売上の適切な期間計上 → C-001で注文日確認
- **R-006**: 売上認識時点の適切性 → C-006で出荷日と計上日の照合

### 表示開示（Presentation & Disclosure）
- **R-001**: 売上の適切な表示 → C-001で取引分類確認
- **R-006**: 売上の適切な開示 → C-006で売上分析
- **R-007**: 債権の適切な表示 → C-007で請求書表示確認
- **R-009**: 売掛金の適切な開示 → C-009で残高管理

### 不正リスク（Fraud Risk）
- **R-001**: 意図的な架空売上計上の不正リスク（収益認識操作）
